# 배포


##
지금까지 우리가 만든 파일은 우리 컴퓨터에 저장되어 있습니다.
이 페이지를 다른 사람이 볼 수 있도록 하려면 어떻게 해야 할까요?


## Netlify에 배포
우리는 [Netlify](https://www.netlify.com/) 라는 서비스에 우리 웹사이트를 올려서 다른 사람들도 볼 수 있도록 할 겁니다.


### Github 계정 생성
Netlify는 Github계정으로 로그인이 가능합니다.

Github은 코드 저장소 제공 서비스이자 개발자들의 커뮤니티 입니다.
Github은 이후에도 사용할 예정이니 계정이 없으신 분은 이참에 만들어보세요.


### 진짜 배포
우리가 만든 index.html 가 존재하는 디렉토리를 아래 화면에 드래그 해서 올립니다.

![스크린샷]()

드래그를 하고 잠시 기다리면 접근 가능한 주소가 발급됩니다.
이 주소로 들어가 내용을 확인해 봅시다.

![스크린샷]()

이제 이 주소를 이용하면 다른 컴퓨터에서도 접근이 가능합니다.


## 사이트 설정
사이트 설정에 들어가면 발급된 주소를 변경할 수 있습니다.
개인 도메인이 있으신 분은 자신의 도메인을 연결할수도 있습니다.
