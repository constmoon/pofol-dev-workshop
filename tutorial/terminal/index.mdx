
# 터미널

터미널은 명령어를 입력 받아 그 결과를 보여주는 프로그램 입니다.

Mac에는 Terminal 이라는 이름으로 Windows에는 Cmd라는 이름으로 들어있습니다.

많은 개발 도구들은 터미널에서 실행하도록 만들어져 있습니다.
개발을 하기 위해서는 이 터미널과 친해져야 합니다.

이번 파트에서는 명령어를 몇가지 실행하며 터미널과 친해져 보도록 하겠습니다.


## Mac
Spotlight에서 Terminal 을 찾아 실행합니다.

명령어 ```date``` 를 입력하고 엔터를 눌러 실행해봅니다. 
현재의 시간을 보여주는 것을 확인할 수 있습니다.


### 현재 위치 - pwd
```
pwd
```

터미널은 Finder와 비슷하게 언제나 현재 폴더를 가지고 있습니다.
위 명령어로 실행되고 있는 현재 폴더를 알 수 있습니다.


### 열기 - open
```
open .
```

터미널에서 ```.```은 현재 디렉토리를 뜻합니다.
```open .``` 는 현재 위치를 연다는 뜻입니다.

위 명령을 실행하면 Finder가 열리는걸 볼 수 있습니다.
```pwd``` 에서 본 사용자 디렉토리가 열린것을 볼 수 있습니다. 

### 파일 목록 - ls
```
ls
```

다시 터미널로 돌아와 ```ls``` 를 입력하면 Finder에서 본 것과 동일한 파일 목록을 볼 수 있습니다.     

### 위치 변경 - cd
```cd```는 터미널의 현재 디렉토리를 변경하는 명령입니다.
 
아래 명령어를 입력하여 바탕화면의 파일 목록을 확인해보세요.
```
cd Desktop
pwd
ls
```

아래 명령어를 입력하여 다시 사용자 폴더로 되돌아 옵니다.
```
cd ..
pwd
```

터미널에서 ```..```은 상위 디렉토리를 뜻합니다.


## Windows
CMD는 명령어를 입력 받아 그 결과를 보여주는 프로그램 입니다.

시작 버튼을 눌러 CMD를 입력해보세요.
까만 아이콘의 CMD를 놀러서 실행해봅니다.

```whoami```
명령어를 입력해보세요.


###
```
explorer .
```
위 명령을 실행하면 현재 디렉토리에서 파일 탐색기를 엽니다.
명령어 마지막에 .이 있는것과 explorer와 . 사이에 스페이스가 있는걸 주의하세요.


### 파일 목록 - dir
```
dir
```
dir은 Directory의 축약으로 현재 폴더의 내용을 보여주는 명령어 입니다.
디렉토리(Directory)는 폴더(Folder)와 동일한 뜻입니다.
탐색기에서 봤던 파일 목록과 동일한 목록을 볼 수 있습니다.


### 위치 변경 - cd
```cd```
아래 명령어를 입력하여 바탕화면의 파일 목록을 확인해보세요.

```
cd Desktop
dir
```
바탕화면의 내용들이 보이시나요?

아래 명령어를 입력하여 다시 사용자 폴더로 되돌아 옵니다.
```
cd ..
dir
```

터미널에서 ```..```은 상위 디렉토리를 뜻합니다.
