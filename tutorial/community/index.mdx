

# 커뮤니티
개발 공부는 단기간에 하기 어렵습니다.
수 많은 자료에도 불구하고 많은 시행착오와 막힘을 경험하게 됩니다.

이럴때 무언가를 공부하며 비슷한 고민을 가지고 비슷한 수준을 가진 학습 동료는 매우 소중한 존재입니다.

내가 막힌 부분을 먼저 겪고 해결했던 경험을 한 사람이 있을 수 있고
다른 사람이 내가 이해한 부분을 어려워할 수 있습니다.

아래 채팅방은 갯츠비 뿐 아니라 개발을 처음 배우면서 겪는 고민들을 나눌 수 있는 공간입니다.
'말 해도 되나' 고민이 되는 순간 자신있게 질문해주세요.
분명 같은 고민했거나 고민하거나 고민할 사람이 있을겁니다.
