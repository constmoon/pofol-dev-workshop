

## 디렉토리 구조
Gatsby 프로젝트를 처음 만들면 디렉토리 안에 이런 파일들이 보일겁니다.

```
src/pages/index.js
package.json
...
node_modules
```


### package.json 
프로젝트의 이름, 설명, 버전과 같은 정보와 사용하고 있는 라이브러리를 담고 있는 파일입니다.
이 파일을 열어 프로젝트 이름과 설명을 적절한 값으로 바꿔봅시다.
```

```

### src/pages/index.js
모든 소스 코드는 ```src``` 폴더 안에 저장됩니다.
그 중에 페이지에 관련된 코드는 ```pages``` 폴더에 저장되며 기본 값으로 index.js 파일이 만들어져 있는걸 볼 수 있습니다.

우리는 이 index.js 파일을 수정하며 사이트를 만들어 나갈 겁니다.
