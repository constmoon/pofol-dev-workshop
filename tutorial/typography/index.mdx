## 타입페이스 - font-family
```font-family```는 여러개를 입력할 수 있습니다.
입력된 폰트 목록은 앞에서 부터 하나씩 사용할 수 있는지 체크합니다. 그리고 컴퓨터에 설치되어 사용할 수 있는 경우 해당 폰트를 이용해 화면을 표시합니다.

그리고 아무것도 매치가 되지 않는 경우 시스템의 기본 폰트를 사용해 글자를 표시합니다.
그래서 마지막에 ```serif``` 혹은 ```sans-serif```를 넣는것이 권장됩니다.


## 웹폰트 - @font-face
컴퓨터에 미리 설치된 폰트를 사용하는 대신 직접 제공한 폰트를 사용하는 방법도 있습니다.
이렇게 제공하는 폰트를 웹폰트 라고 부릅니다.

웹폰트는 직접 제공하거나 기존 서비스를 이용해 제공하는 방법이 있습니다.
여기서는 더 간단한 기존 서비스를 이용하는 방법을 써보겠습니다.

웹폰트는 제공하는 서비스 마다 사용 방법에 약간씩 차이가 있습니다.
크게는 정의하고 사용하는 부분으로 이루어져 있고 정의하는 부분은 대부분의 서비스에서 복사해서 사용할 수 있도록 코드를 제공해줍니다.


### 유형별 사용법

가장 흔한 ```@font-face```를 이용하는 방법입니다. ```@font-face``` 안에 src 속성으로 웹폰트 파일의 주소를 지정하면 브라우저는 해당 파일을 받아옵니다.
그러면 ```@font-face```안에서 지정한 이름으로 웹 폰트를 사용할 수 있습니다. (바로 아래 p 선택자를 보세요.)

```
@font-face {
    font-family: 'NanumBarunGothic';
    font-style: normal;
    font-weight: 400;
    src: url('//cdn.jsdelivr.net/font-nanumlight/1.0/NanumBarunGothicWeb.woff') format('woff');
}

p {
    font-family: 'NanumBarunGothic';
    font-weight: 400;
}
```


```@import```문을 사용하는 경우. ```@import```는 ```<style></style>``` 태그 안 이나 .css 파일의 내부에 넣어 사용합니다.
```
<style>
@import url(//fonts.googleapis.com/earlyaccess/nanumgothic.css);

p {
    font-family: 'Nanum Gothic', sans-serif;
}
</style>
```

구글 웹폰트 서비스는 ```<link>``` 태그를 제공합니다. ```<link>``` 태그는  ```<head />```태그 안에 넣어 사용합니다.
```
<html>
<head>
    <link href="https://fonts.googleapis.com/css?family=Noto+Sans+KR&display=swap" rel="stylesheet">
    ...
</head>
...
```
