
# Gatsby사이트의 배포

## Build
우리 프로젝트에 .html 파일이 없다는걸 눈치 채셨을 겁니다.
현재 작업하고 있는 코드가 담긴 폴더를 그대로 Netlify에 드래그 해도 웹사이트가 정상적으로 동작하지 않는걸 알 수 있습니다.

Gatsby로 만든 사이트를 배포 하기 위해서는 먼저 index.html 파일을 포함하여 웹브라우저가 바로 해석할 수 있는 파일들을 생성하는 과정이 필요합니다.


```
npx gatsby build
```
위 명령을 실행하면 현재 폴더에 ```public```이라는 폴더가 새로 만들어진걸 볼 수 있을겁니다.

이 폴더의 내용을 보면 ```index.html```파일을 포함하여 여러 파일이 생성된걸 볼 수 있습니다.

바로 이 폴더를 Netlify에 업로드하면 이제 우리가 만든 사이트가 나오는걸 확인할 수 있습니다.
