
# 설치

이번 파트에서는 이번 워크샵에서 사용하게 될 개발 도구를 설치하겠습니다.
설치는 앞서 살펴본 터미널에서 진행합니다.


## Mac

### Homebrew
Homebrew는 Mac에서 프로그램을 설치해주는 프로그램입니다.
Homebrew를 사용하면 설치한 프로그램의 설치, 업그레이드, 제거를 간단하게 할 수 있습니다. 

[Homebrew 웹사이트](https://brew.sh/index_ko)
웹사이트에 나와있는 설치 명령을 복사해 터미널에 붙여넣으면 설치가 됩니다.

### 설치
방금 설치한 brew를 이용하여 개발에 필요한 도구들을 설치합니다.
```
brew install git
brew install node
brew cask install visual-studio-code
brew cask install github
```
혹시 이전에 설치한 분이 계시다면 너무 걱정하지 말고 설치 되지 않은것만 설치 하시면 됩니다.


### 크롬
혹시 크롬이 설치되어있지 않다면 크롬도 설치합니다.
```
brew cask install google-chrome
```


## Windows

### Choco
Chocolatey는 프로그램을 설치해주는 프로그램 입니다.
Chocolatey로 설치하면 설치된 프로그램 제거, 업데이트를 간단하게 할 수 있습니다.


1. CMD를 관리자 권한으로 엽니다.
[스크린샷]()

2. CMD 안에서 ```powershell``` 을 실행합니다.

3. https://chocolatey.org/install#individual 페이지의 "Now run the following command:" 의 내용을 복사해 붙여넣습니다.


### 설치
```
choco install git
choco install nodejs
choco install vscode
choco install github-desktop
```

### 크롬
혹시 크롬이 설치되어있지 않다면 크롬도 설치해줍니다. (설치되어 있으면 하지 않으셔도 됩니다.)
```
choco install GoogleChrome
```

## Ready!

이제 필요한 도구를 모두 설치했습니다.
제대로 설치가 되었는지 Visual Studio Code를 실행해봅니다.

Mac에서는 Spotlight에서 Windows에서는 시작 버튼에서 Visual Studio Code를 찾을 수 있습니다.
