
## 레이아웃
지금까지 만든 HTML 파일을 열면 다음 처럼 보일겁니다.

레이아웃은 한 요소의 크기를 지정하는 부분과 그 요소를 배치하는 부분으로 나눌 수 있습니다.


## 박스 모델 Box Model
요소의 가로와 세로의 크기를 지정해보겠습니다.

### width, height
```
<div style="width: 300px; height: 200px; border: 2px solid black;">
BOX
</div>
```
<div style="width: 300px; height: 200px; border: 2px solid black;">
BOX
</div>


### padding, margin
<div style="border: 2px solid black">
    <div style="
        width: 300px;
        height: 200px;
        border: 2px solid black;
        padding: 20px;
        margin: 20px;
    ">
    BOX
    </div>
</div>

### box-sizing
<div style="
    width: 300px;
    height: 200px;
    border: 2px solid black;
    padding: 20px;
    margin: 20px;
    box-sizing: border-box;
">
    BOX
</div>


## 특수한 태그 div
div는 그 자체로 눈에 보이지는 않고 문서의 부분을 나눠주는 태그입니다.
특히 배치를 위해 페이지를 나눌 때 용이합니다.

```
<div>

</div>
``` 
 
## 의미에 맞는 (Semantic) 태그들
div와 동작은 동일하지만 다른 이름을 가진 태그들이 있습니다.
```
<header>
    <nav></nav>
</header>
<main>
    <section>
    </section>
</main>
<aside>
</aside>
```
이러한 태그들을 이용하면 좀 더 의미에 맞는 코드를 작성할 수 있습니다.


## Flex Layout
Flex는 페이지를 배치하는 방법 중 하나 입니다.

```
display: flex;
```

방향
```
flex-direction: column;
flex-direction: row;
```


```
justify-content: center;
justify-content: space-between;
justify-content: space-around;
```

```
align-items: center;
align-items: stretch;
```


```
flex-grow: 1;
flex-shrink: 1;
```
