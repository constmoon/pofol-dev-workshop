
## 시작
터미널에서 아래 명령을 입력합니다.

```
npx gatsby new my-portfolio https://github.com/pofoldev/pofol-dev-starter
```

```npx```는 자바스크립트 패키지가 가지고 있는 명령어를 실행하는 명령어 입니다.
위 명령을 실행하면 ``gatsby``` 명령이 들어있는 ```gatsby-cli```를 설치한 후 ```gatsby new``` 명령어를 실행합니다.


## 갯츠비 스타터
갯츠비는 [스타터](https://www.gatsbyjs.org/starters/?v=2)라는 것이 있습니다.
스타터를 이용하면 처음부터 웹사이트를 만들 필요 없이 어느정도 만들어진 사이트를 수정하는 방법으로 사이트를 만들 수 있습니다.

우리는 방금 pofol-dev-starter 라는 스타터를 기반으로 프로젝트를 만들었습니다.
우리가 사용한 이 스타터는 학습을 위해 최소한의 구성만 갖춰놓은 스타터 입니다.


## 개발
이제 생성된 사이트를 실행해 봅시다.
```
cd my-portfolio
npx gatsby develop
```

```gatsby develop```은 개발을 할 수 있도록 gatsby로 만들어진 웹사이트를 실행하는 명령어 입니다.

실행 결과를 보면 http://localhost:8000 에서 접근이 가능하다는 메세지가 보입니다.
